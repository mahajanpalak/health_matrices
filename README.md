# Health Matrices Dashboard

**Health Matrices** is an interactive Python application that helps users **track and analyze their health metrics** such as BMI, weight status, and other personal data. The project focuses on creating a **user-friendly dashboard** that provides **real-time calculations** and visual insights based on user input.

## Features

- **User Profile Management:** Create and update user profiles with details like name, age, height, and weight.  
- **Real-Time Calculations:** Calculates BMI and weight status instantly.  
- **Interactive Dashboard:** Visualizes health metrics using charts and graphs.  
- **Data Persistence:** Stores user data in CSV files for easy retrieval and updates.  
- **Dynamic Insights:** Highlights trends and provides actionable health insights.

---

## Technologies Used

- **Programming Language:** Python  
- **Data Handling & Analysis:** Pandas, NumPy  
- **Visualization & Dashboard:** Plotly, Streamlit  
- **Data Storage:** CSV files  
- **Version Control:** Git, GitHub
